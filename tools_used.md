## Tools Used
This document outlines the tools used during the vulnerability assessment of https://karteverything.vercel.app as part of the Future Interns Cyber Security Task 1.  

### Tool Overview
| Tool | Version | Purpose |
|------|---------|---------|
| OWASP ZAP | 2.17.0 | Passive vulnerability scanning and security analysis |
| Nmap | 7.99 | Network exposure and open port identification |
| Browser Developer Tools | Latest | Inspection of HTTP headers, cookies, and client-side resources |
| curl | Built-in | Verification of HTTP response headers |

### Conclusion
The tools used during this assessment provided sufficient visibility into the application's public-facing security posture while remaining within the ethical and technical scope defined for the project.
